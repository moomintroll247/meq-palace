# MEQ Memory Palace

A 3D Myst-style walkable memory palace for RANZCP MEQ 2026 exam revision.
Twenty themed islands of psychiatry curriculum, rendered procedurally in Three.js,
served as a single HTML file.

Live: <https://moomintroll247.github.io/meq-palace/>

## Run locally

```
node server.js   # http://localhost:5181
```

## Controls

**Desktop**: click to enter, `WASD` to walk, mouse to look, `E` to interact,
`Shift` to slow, `Esc` to release.

**iPad / touch**: tap to enter, left-thumb joystick to walk, drag the right
half of the screen to look, on-screen `E` button to interact. Add to Home
Screen for fullscreen launch.
